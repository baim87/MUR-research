Understanding how Tiktok works.

Tiktok have two different entity and database source.

  • 1. Tiktok
  • 2. Douyin
Tiktok will serve any request from outside China mainland. While Douyin only will serve request from China users. Presently Tiktok and Douyin have different databases, and it won't connect or serve content from another platform. So, if we want to targetting people that speaking Chinese, this is crucial to understand. No, matter what language the user are using, the content always served from the user GEO location.

How the FYP and search works? For You page will be limited to the local satisfaction metrics per category that the user want to watch. Each metrics have layers of satisfaction, the satisfaction metrics are:
  • Count of Views
  • Count of time of being watched
  • Count of Likes
  • Count of Share
When these metrics been satisfied, then it will be recommendation video that will be shown on the For You section. Each categories are competing based on these metrics to earn a spot on local then global FYP. i.e category travel. If tiktok thinks that #1 recommendation of Travel category or hashtag on Japan will be relevant to the user that wanted to watch Travel Japan from USA Geo, they will serve them. N.B the priority are local recommendation 1st if there's none, they will show the recommendation from other geo base video that relevant to the user.

Proof.

I'm using VPN to access tiktok, the results are different based on the users geography. When I'm login from Indonesia, the FYP will be showing local indonesia. When I'm login from Japan, the FYP will be showing local Japan, etc. This will cause problems, if we're trying to upload video from Japan that targetting people overseas, even if we're using content localization to target specific country. This will make our video have to competing in the local 1st before it can be recommended to the global FYP or search.

Solution.

To answer this issue, we need to upload localization video using VPN. i.e. If we would like to target USA, use USA VPN to upload the content so they will be stored to local USA database. If we're targetting Vietnam, use VPN Vietnam so they will be stored to local Vietnam. But if we would like to target China, we need to create Douyin account and upload the video to the Douyin account using China VPN.
